# ☕ Oficina da Xícara

Uma loja virtual moderna desenvolvida com **HTML, CSS e JavaScript puro**, focada na venda de **canecas, xícaras e copos personalizados**.

O projeto foi construído para ser hospedado gratuitamente no **GitHub Pages**, oferecendo uma experiência rápida, responsiva e agradável para o usuário.

---

## ✨ Funcionalidades

### 🛍️ Catálogo de Produtos

- Exibição dinâmica dos produtos
- Múltiplas fotos por produto
- Suporte a vídeos dos produtos
- Navegação entre mídias
- Busca em tempo real
- Filtro por categoria

### 🛒 Carrinho de Compras

- Adicionar produtos ao carrinho
- Alterar quantidades
- Remover produtos
- Limpar carrinho
- Cálculo automático do valor total
- Persistência utilizando LocalStorage

### 📱 Integração com WhatsApp

Ao finalizar a compra, o sistema gera automaticamente uma mensagem contendo:

- Produtos selecionados
- Quantidade de cada item
- Valor total da compra

e abre uma conversa diretamente no WhatsApp.

### 🎨 Galeria de Artes

- Exibição das artes já produzidas
- Carregamento progressivo das imagens
- Lazy Loading
- Hover Effects Premium
- Layout responsivo

### 🌙 Dark Mode

- Alternância entre tema claro e escuro
- Preferência salva automaticamente no navegador
